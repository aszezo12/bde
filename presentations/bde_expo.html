BDE Expo, January 24-26, 2023¶

The BDE Expo is a multi-day event featuring presentations by the BDE team between January 24-26. These sessions will describe the direction of future BDE development, as well as providing a forum to answer questions and listen to feedback. We plan to highlight interesting technical areas and engineering solutions that we hope are valuable to the C++ community at Bloomberg.

Most of the sessions will be held at 919 3rd Ave, room 16-7, with remote access provided via Zoom. If you can, please join us in person!

BDE Roadmap: 2022, 2023 and Beyond (and BDE Expo Intro)¶

The presentation will highlight features that have been released recently, survey projects that are in development, as well as discuss a high-level vision for how the BDE libraries (and BSL in particular) will evolve over the next couple years. The session will provide substantial time at the end for questions and discussion.

ELRN: BDE Roadmap: 2022, 2023, and Beyond (and BDE Expo Intro)
Slides: 2023-01-24_BDE_Roadmap.pdf
Time: Tue, Jan 24, 2023, 10:00 AM - 11:00 AM EST
Presenter: Henry Verschell

Qualifying Queue Quirks: Towards a Common Concurrent Queue Interface¶

While all concurrent queues in BDE roughly share most basic operations, the interfaces are far from standardized, causing issues when switching from one implementation to another. Ideally, one queue should be a drop-in replacement for another, as long as that queue is suited for that scenario (e.g., a single producer queue can be used only when there is a single producer). To reach this goal, a common interface for queues is being defined and implemented in the latest additions to the BDE family of concurrent queues in bdlcc. This common interface addresses some recurring “pain points”, exposes required functionality for higher-level BDE components, makes changes to the most basic operations, and removes quirky functionality available in isolated implementations. Future additions to this interface are also discussed, with the hope of gauging the level of user interest in the features.

ELRN: Qualifying Queue Quirks: Towards a Common Concurrent Queue Interface
Slides: 2023-01-24_Qualifying_Queue_Quirks.pdf
Time: Tue, Jan 24, 2023, 11:30 AM - 12:30 PM EST
Presenter: Jeffrey Mendelsohn

Modern Allocator Use at Bloomberg¶

This session will introduce a new, more modern pattern for developing allocator-aware software. BDE has provided support for writing allocator-aware software since its inception. In 2023 we will release the next generation of allocator support, incorporating recent changes to the C++ standard, such as std::pmr::polymorphic_allocator. In this session we will demonstrate how we have consolidated C++ standard support for allocators into BDE, enabling users to write software that is compatible with both the standard and already existing Bloomberg software. We will also preview our upcoming book on allocators, C++ Allocators for the Working Programmer, and discuss future advancements we hope to bring to the standard.

ELRN: Modern Allocator Use at Bloomberg
Slides: 2023-01-25_Modern_Allocator_Use_at_Bloomberg.pdf
Time: Wed, Jan 25, 2023, 10:00 AM - 11:00 AM EST
Presenter: Joshua Berne

Tour of the BHL HTTP/2 Library¶

The BHL library provides an asynchronous C++ API for HTTP/2 communication. In this presentation, we will cover the basics of the HTTP/2 protocol, its core strengths, and how the BHL library enables creating HTTP/2 servers and clients. We will discuss some of the key design decisions in BHL and demonstrate the BHL library in action with example server and client applications. Finally, we will touch on future development of this library, including changing the event model and incorporating HTTP/2 flow control.

ELRN: Tour of the BHL HTTP/2 Library
Slides: 2023_01_24_Tour_of_the_BHL_HTTP2_Library.pdf
Time: Wed, Jan 25, 2023, 11:30 AM - 12:30 PM EST
Presenter: Rostislav Khlebnikov

BALL Logging: How to get the Logs You Want¶

Writing to a log is easy, but configuring a logger optimally can be hard. In this presentation we will discuss the array of logging configuration options and provide recommendations for their use. We will also discuss some more recent enhancements to the logger, such as log attributes and JSON record formatting. Finally, we will present our plans for 2023.

ELRN: BALL Logging: How to get the Logs You Want
Slides: 2023_01_26_Ball_Logging.pdf
Time: Thu, Jan 26, 2023, 10:00 AM - 11:00 AM EST
Presenter: Oleg Subbotin

Visualizing Code at Scale¶

BDE has been working with Codethink to produce a powerful visualization tool capable of viewing the physical and logical design of large C++ codebases. This visualization is based on the methodology described in John Lakos’s Large-Scale C++ Volume I. Join us as we demonstrate how the tool renders large BDE-style repositories, promotes the design of well-constructed component-based software, and enables the visualization of modifications to existing systems. Leave with an invitation to beta test our product and provide valuable feedback on how it evolves from here.

ELRN: Visualizing Code at Scale
Slides: N/A
Time: Thu, Jan 26, 2023, 11:30 AM - 12:30 PM EST
Presenter: Harold Bott, Joshua Berne

For more information, please join the “Code Visualization Diagram Tool” IB Forum, or IB Joshua Berne or Harold Bott
