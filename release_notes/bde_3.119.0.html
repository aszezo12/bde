BDE 3.119.0 Release¶

Schedule¶

  • The BDE team announces that the BDE 3.119.0 production release was completed on Monday, Jun 5, 2023.

BDE 3.119.0 Highlights¶

C++20 Library Features¶

For compilers that support the 3-way comparison operator <=>, added support for the following types:

  • bsl::shared_ptr

Added the header <bsl_coroutine.h>, which provides the following aliases for C++20 types related to coroutines:

  • bsl::coroutine_traits

  • bsl::coroutine_handle

  • bsl::noop_coroutine

  • bsl::noop_coroutine_promise

  • bsl::noop_coroutine_handle

  • bsl::suspend_never

  • bsl::suspend_always

The functions bsl::make_shared` and bsl::allocate_shared now have overloads for array element types.
